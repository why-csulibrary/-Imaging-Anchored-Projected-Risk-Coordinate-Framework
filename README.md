This repository provides a simplified demonstration of our workflow for interpreting an imaging-anchored projected risk coordinate (μ) across external multi-omics data layers in metastatic hormone-sensitive prostate cancer. The underlying study used TTCR as a clinically anchored task and examined how a fixed, baseline-computable risk coordinate could be related to bulk transcriptomic signals, single-cell-informed cell states, spatial tissue organization, and cfDNA copy-number readouts. This public version is intended to illustrate the analytical logic only and is not the exact codebase used for the full manuscript analyses.
